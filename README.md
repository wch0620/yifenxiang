## 技术相关
### 一、 后台用到的技术
  * 1.1 spring框架：springframework4.3.7.RELEASE
  * 1.2 数据库框架：mybatis3.4.1
  * 1.3 安全验证模块：shiro1.3.2
  * 1.4 数据库：mysql8.0.12
  * 1.5 缓存框架：j2cache2.3.22-release
  * 1.6 验证码：kaptcha2.3.2

### 二、 小程序端用到的技术
 * 1.1 Vue
